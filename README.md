# EE469 - Digital Design Labs
**University of Washington | Spring 2026**

This repository contains lab implementations for EE469, where we incrementally build a pipelined ARM 64-bit microprocessor using structural Verilog. All datapath logic is gate-level (AND, OR, NAND, NOR, XOR, etc.) with no behavioral RTL. Development and simulation is done using Quartus II and ModelSim.
